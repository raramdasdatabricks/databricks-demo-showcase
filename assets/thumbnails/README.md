# Drop your card thumbnails / video posters here

Optional. Put a still image (`.jpg`/`.png`/`.webp`) per demo and reference it via the
`thumbnail` field in `data/demos.json`. It's used as both the card preview and the
video poster frame.

- **Aspect ratio:** 16:9 (e.g. 1280×720). Anything off-ratio is center-cropped.
- **No thumbnail?** No problem — cards fall back to a clean, branded category gradient,
  so the site looks finished even before you add any images.
